# 模型一：综合总结（新版框架）

**目标**：在无法观测粉丝票的前提下，恢复粉丝意愿并量化赛制公平性。  
**方法**：硬约束反演 + 截断贝叶斯 MCMC + 反事实评估 + 机制设计。

---

## 核心贡献
1. **逻辑闭环**：规则一致性作为硬约束，避免“用结果解释结果”。
2. **概率恢复**：在 LP/MILP 区间内构建后验分布，输出 HDI 区间而非单点。
3. **反事实评估**：可替换规则引擎，量化 Kendall Tau / 逆转率 / 孔多塞一致性。
4. **机制设计**：动态权重 + 帕累托前沿，权衡公平性与参与度。
5. **特征解释**：XGBoost + SHAP 捕捉非线性与交互关系。

---

## 主要输出
- 规则透明度审计图（揭示规则黑箱周次）。
- 后验带状图（如 Bobby Bones 案例）。
- 帕累托前沿（动态权重机制的“甜蜜点”）。
- SHAP 重要性与年龄依赖图。

---

## 结论要点（模板）
- 多数赛季满足规则一致性，少数周次显示规则不透明。
- 动态权重机制在公平性与观众影响之间取得更佳平衡。
- 关键特征排序更接近“舞伴 + 评委分 + 年龄”而非单一知名度。

---

## 建议
- 节目组可在赛季开始前运行一致性审计，提前标记高争议周次。
- 建议公开评委拯救规则或量化指标，以降低“黑箱”质疑。
